  • IMAGE PROCESSING : WebMail CAPTCHA prediction [ONGOING]

CAPTCHA is an acronym for "Completely Automated Public Turing test to tell Computers and Humans Apart". It is actually a reverse Turing test, because it is administered by a machine and targeted at a human, in contrast to the standard Turing test that is typically administered by a human and targeted at a machine.
The popular mail servers like Yahoo!®, Google® use Captchas like the ones shown below either make segmentation difficult is to crowd symbols together or by adding an angled line. In either of these cases, the spacing between the characters, as well as the size of the characters themselves are not fixed, they vary from one to another. Moreover, they are indented arbitrarily, making it difficult to separate them.
The CAPTCHAs used by the IITG WebMail (provided by Squirrel Mail), the text is aligned perfectly horizontally, with a uniform spacing in between characters, which renders them comparatively more prone to attacks.
  • SPEECH PROCESSING : An Isolated Word Recogniser based on Vector Quantization

Vector Quantization is a decent way to identify words from a speech and then further make use of it. The first task involved in doing this is to collect a pool of sample data to train the system. Since this is targeted at recognising isolated words, we need to have a data set of the discrete words spoken. For extracting the vectors, we choose a window of 20 ms and shift this window of 20ms forward every 10ms. The data recorded from each frame is processed to get the Melfrequency Cepstral Coefficients (MFCC). Now, to represent the feature of a frame, the first 39 coefficients are chosen. VQ based IWR makes use of an algorithm known as K-means algorithm which involves finding clusters in a set of data and identifying a group of data by the cluster to which it belongs. In K-mean clustering a set of random vectors are chosen and are assigned means at first iteration which gets refined to original mean after successive iterations. The number of iteration is decided by the convergence of mean and finding the dissimilarity between previous and current means. Once the means have been generated from the training set of data they can be called codewords for a particular class that will be used in identifying the new word. This process is repeated for all the different sets and codewords are generated respectively for each class.
To identify a new word the task involved is extraction of feature vector from each frame of the spoken word and then find the distance of each of the testing data feature from every single codeword and label the minimum distance codeword. Once it is done the sum of distance of each of the minimum distance codeword is found and is compared with the sum of distance of codewords of each class. The one which yields the minimum difference is chosen to be the class to which the testing data belongs.
The final code was written in MATLAB, and this is the link to the MATLAB file (.m) that creates clusters out of the several data points obtained from the dataset(s).
